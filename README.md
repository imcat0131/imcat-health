## this is fitbit api sample program

### refarence

https://dev.fitbit.com/build/reference/web-api/

### how to use

1. get access token and refresh token

2. set access token and refresh token to test_conf.json

```json
{
  "client_id": "",
  "access_token": "",
  "refresh_token": ""
}
```

3. run sample program

```bash
$ python3 script.py
```
